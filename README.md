# DBSeeder
A Project made at GDGC's Hackathon, PPSU Techsprint
with contributions of [Dharmik Chokhaliya](https://github.com/Dharmik-Chokhaliya), [Varshil Nasit](https://github.com/Varshil15), and [Khush Patel](https://github.com/khushpatel00/)

This project phrases the database tables, and add mocked data, that can be utilized for testing, or in-development projects as well

To run this Project locally, open the main Branch in you IDE, or code editor and use this command

```
npm start
```



if you encounter any issues while executing the command, remove the ```node_modules``` folder, and run this command

```
npm i
```

& then

```
npm start
```
